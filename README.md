# OIBSIP_WDD_2
Objective:
The objective of this project is to create a simple and meaningful tribute page dedicated to Dr. A.P.J. Abdul Kalam. This project helps in practicing basic web development skills while also presenting information in a clean and respectful way.

What I Did (Steps Performed):
Created the structure of the webpage using HTML
Added a meaningful quote and image of Dr. Kalam
Wrote a short biography section highlighting his life and achievements
Styled the page using CSS (external stylesheet)
Organized the content into sections like:
Title
Image
Biography
Footer

🛠️ Tools & Technologies Used
HTML5 – For structuring the content
CSS3 – For styling the webpage
VS Code – Code editor
Git & GitHub – Version control and hosting

💡 Features:
📖 Clean and readable layout
🖼️ Image with a motivational quote
🧑‍🚀 Short biography of Dr. A.P.J. Abdul Kalam
❤️ Simple and elegant design
📱 Responsive structure

📊 Outcome:
Through this project, I learned:
How to structure a webpage properly
How to present content in a visually appealing way
Basics of combining HTML and CSS
Importance of simplicity in design
It also helped me improve my understanding of creating real-world beginner projects.

Final Thoughts:
This project is a small tribute to one of India’s greatest personalities, Dr. A.P.J. Abdul Kalam. It not only helped me practice coding but also reminded me of his inspiring life and achievements.
